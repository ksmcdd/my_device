本地数据与备份
Storage

本地存储：学习记录、任务池、周记录和界面偏好都保存在当前浏览器的 localStorage 中，不会自动上传到服务器。

导出 / 导入：Dashboard 右侧的“导出备份 / 导入备份”会读写 JSON 文件。导入会覆盖当前浏览器中的现有数据，建议先导出一份。

清空本地数据：只影响当前浏览器当前站点的记录，不会删除仓库文件。
