Guide / Local Data Notes

说明与本地数据

这一版仍然只有一个 HTML 主文件，不需要 Node、npm、React 或 Vite，也不需要安装依赖。直接双击或用 Live Server 打开即可。
步骤你要做什么原理
1把当前页面保存为 index_v4.html浏览器可直接识别并打开单文件页面
2用 Live Server 或双击打开浏览器会执行页面里的 CSS 和 JS
3上传到 GitHub Pages 或静态托管页面代码可公开访问，但本地数据仍在浏览器里
4定期导出备份文件避免清缓存、换浏览器或换电脑后丢失记录
关键点：
1. 公开的是网页代码，不是你的学习数据。
2. 每个用户打开同一网址，看到的是自己浏览器里的本地数据。
3. “清空本地数据”只会删除当前浏览器中的记录，不会影响别人。
